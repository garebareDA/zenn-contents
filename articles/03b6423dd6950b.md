# Cicaとは
Cica（シカ）は、プログラミングでの利用に最適化された日本語等幅フォントです。
英数字や記号には「Hack」や「DejaVu Sans Mono」、日本語には「Rounded Mgen+」、絵文字に「Noto Emoji」などが採要された合成フォントです。
https://github.com/miiton/Cica

等幅フォントは英数字と日本語が交じると端が揃わなかったり、視認性が悪いなどの状態となることがあるため英数字・日本語の幅を揃えたものです。
英数字の幅が1, 日本語の幅が2となっています。

# Cicaの良いところ
- 高い視認性: 濁点・半濁点の判別がしやすく、0（ゼロ）とO（オー）、1（いち）とl（エル）などの見間違えやすい文字が区別しやすくなっています。
![](/images/Cica/image-1.png)
- 非HiDPI（非Retina）のWindows環境でも文字が欠けにくいよう調整されています。
- ターミナルでよく使われるアイコン（Nerd Fonts相当）や、tmuxの画面分割線などにも対応しています

## 独自の調整
-「m」の調整: 小文字の m の真ん中の棒をあえて短くすることで、rn（アール・エヌ）との見間違えを防ぎつつ、少し可愛らしい印象に仕上げています。
![](/images/Cica/image.png)

# Nixでインストールする
私はNixでCicaフォントをGitHubからインストールしています。
```nix
...
let
  cica = pkgs.stdenv.mkDerivation rec {
    pname = "cica";
    version = "5.0.3";
    src = pkgs.fetchurl {
      url = "https://github.com/miiton/Cica/releases/download/v${version}/Cica_v${version}.zip";
      sha256 = "...";
    };
    nativeBuildInputs = [ pkgs.unzip ];
    sourceRoot = ".";
    installPhase = ''
      mkdir -p $out/share/fonts/truetype
      cp *.ttf $out/share/fonts/truetype/
    '';
  };
...

fonts.packages = [ cica ];
```

# 個人的感想
Noto Emojiが採用されているので、Zennを書いている時に絵文字のフォントがエディタに馴染むのが結構嬉しいです。
太字(Bold)、太字斜体(BoldItalic)、斜頸(RegularItalic)が用意されているのも嬉しいポイントですね、意外と揃っているフォントが少ない。
とりあえずこのフォントをインストールしてエディタで指定しておけば、表示が崩れることはないだろうということで愛用しています。
