# kyとは
ブラウザ、Node.js、Denoをサポートしている`fetch`をベースにした小さくシンプルなHTTPクライアントです。

## リポジトリ
https://github.com/sindresorhus/ky

# インストール
```
npm install ky
```
npm

```
import ky from 'https://esm.sh/ky';
```
deno

## 使い方
```js
import ky from 'ky';

const json = await ky.post('https://example.com', {json: {foo: true}}).json();

console.log(json);
``
